# 예제코드 실행 방법
```bash
bun start
```
- package.json 파일에 scripts에 start 에 실행하고 싶은 파일 경로 추가
- bun start 명령어로 실행


# 폴더구조
- src/ch* : 챕터의 예제 코드
  - original.ts : 책에서 제공하는 예제 코드
  - refactoring.ts : 책에서 리팩토링한 코드
  - my-refactoring.ts : 내가 리팩토링한 코드
  - test.ts : 테스트 코드
  - etc.. : 잡다한 필요한 파일
- src/shared : 공통으로 사용
  - types.ts : 타입 정의
  - utils.ts : 유틸리티 함수
