# typescript

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.1.45. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.


# 예제코드 실행 방법
```bash
bun start
```
- package.json 파일에 scripts에 start 에 실행하고 싶은 파일 경로 추가
- bun start 명령어로 실행
